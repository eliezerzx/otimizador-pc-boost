# 🚀 Otimizador PC Boost

O **Otimizador PC Boost** é um script de automação desenvolvido para Windows que visa melhorar o desempenho do sistema através da limpeza de arquivos temporários, otimização de cache e ajustes de telemetria. 

Este projeto foi criado para usuários que buscam uma manutenção rápida e eficaz sem a necessidade de instalar softwares pesados de terceiros.

---

## ✨ Funcionalidades

* **Limpeza de Temporários:** Remove arquivos das pastas `TEMP`, `%TEMP%` e `Prefetch`.
* **Limpeza de Cache DNS:** Libera e renova o cache do resolvedor DNS para melhorar a conectividade.
* **Otimização de Disco:** Aciona ferramentas nativas para remoção de arquivos inúteis do sistema.
* **Redução de Processos:** (Opcional) Desativa serviços de telemetria desnecessários para liberar memória RAM.
* **Simplicidade:** Execução direta via terminal com privilégios de administrador.
